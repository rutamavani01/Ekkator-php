N.B :
 - BEFORE UPDATING TO THE LATEST VERSION, WE WOULD HIGHLY APPRECIATE IF YOU KEEP A BACKUP OF THE EXISTING VERSION.

 - Before updating to version 1.4, make sure that you currently running your product on version 1.3.

Let's proceed,

- Go to system settings from settings menu in superadmin panel
- Move to the section 'Update Product' on that page
- Upload the file 'update_1.4.zip' which is located at Update pack > v1.4 folder inside your copy of Ekattor8 downloaded from Codecanyon.
- Hit the button 'Update'.
- For version 1.4 update if you face 500 error then don't worry your product is alredy updated. You can check it from your Superadmin panel. Go to Settings > About. You will see your version is updated to version 1.4
- If you encounter any layout related issues, clear your browser cache. Press the following shortcut keys to clear your browser cache.
	a. For Windows: Ctrl + Shift + R
	b. For Mac: Command + Shift + R

- That's it.
